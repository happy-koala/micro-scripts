# Pomodoro Timer

A simple Pomodoro timer in BASH.

- Includes graphical output and audio notification
- Works on macOS


## Usage

Download the file `pomodoro.sh`.


### Execute the file 

Make the script executable:

```bash
chmod +x pomodoro.sh
```

Run the timer:

```bash
./pomodoro.sh
```


### Run from anywhere 

Trim the file extension.

Move the file to `/usr/local/bin/`.

Run from anywhere: 
```bash
pomodoro
```
